The Honors Computer Science Testing program is the most complex and largest program I have ever written. I wrote it completely from scratch, and in the end, it used Typescript with Deno on the backend, Sequelize for the database, and SvelteKit as the frontend.

It has multiple vital systems- a test generation system, authentication, session management, and a SvelteKit frontend.

Skills

I think I actually gained experience in this project. I learned about managing large repositories, first of all. I had to manage some large files and some complex changes to file structure. I also did some major refactoring, and version control came in clutch multiple times.

I also gained lots of skills in the usual full stack web development. I learned about creating robust APIs, and managing complex states like database stuff, sessions, and authentication. I definitely could've done better.
